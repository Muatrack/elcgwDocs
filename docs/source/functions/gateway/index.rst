网络设置
-----------

   网关联网方式有wifi、有线网络、4G(可选), 联网后可通过访问网关的web本地操作设备管理、平台配置等事项。
   
      *通过按下网关上的\ **SET**\ 按钮，可以开启网关的AP, SSID:ws-agw01-ap, PW:123456789*, 可通过电脑/手机的wifi连接实现访问网关web（http://192.168.20.1）。
