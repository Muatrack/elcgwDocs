网关功能-设备管理
===================

id-10000_增加断路器
----------------------

当前用例将为全新的网关增加断路器，目的在于测试网关的基本功能-增加设备是否符合预期。网关中支持断路器的最多数量为32台，在网关的管理逻辑中，将为每一台新增的设备指定ID(网关内唯一)。
网关中所有设备ID不能重复，ID总数为32个，取值范围 [1,32]。

*网关分配给断路器的ID亦是MODBUS协议中的从站地址，因此有些旧版本文档中称之为 地址*
